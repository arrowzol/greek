    short sound: ᾰ[εο]ῐῠ

accents:
    ;       άέίόύήώ     acute (/)
    ]       ὰὲὶὸὺὴὼ     grave (\)
    [       ᾶ-ῖ-ῦῆῶ     circumflex (~)
smooth breathing marks:
    '       ἀἐἰὀὐἠὠ     smooth (s)
    /       ἄἔἴὄὔἤὤ     smooth + acute
    =       ἂἒἲὂὒἢὢ     smooth + grave
    -       ἆ-ἶ-ὖἦὦ     smooth + circumflex
rough breathing marks:
    "       ἁἑἱὁὑἡὡ     rough (r)
    ?       ἅἕἵὅὕἥὥ     rough + acute
    +       ἃἓἳὃὓἣὣ     rough + grave
    _       ἇ-ἷ-ὗἧὧ     rough + circumflex
iota subscripts:
    {       ᾳ----ῃῳ     iota subscript (i)
    {;      ᾴ----ῄῴ     iota + acute
    {]      ᾲ----ῂῲ     iota + grave
    {[      ᾷ----ῇῷ     iota + circumflex
    {'      ᾀ----ᾐᾠ     iota + smooth
    {"      ᾁ----ῇῷ     iota + rough
vowel stuff:
    :       --ϊ-ϋ--     "double dots"
    W       --ΐ-ΰ--     "double dots" + acute
    o-l     ᾰ-ῐ-ῠ--     short vowels (S), keyboard-l for "long"
    s-o-l   ᾱ-ῑ-ῡ--     long vowels (L), keyboard-l for "long"

punctuation (modern):
    ,   break a clause
    .   period drawn low, end a sentence
    ·   dot drawn center, like english (:), "raised dot" types with opt-9
    ;   like english (?)
